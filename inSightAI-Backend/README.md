# 🚀 InsightAI Backend

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Multer
- ExcelJS
- PDFKit

## Installation

```bash
npm install
```

Create `.env`

```env
PORT=5000

MONGO_URI=your_mongodb_connection

JWT_SECRET=your_secret

CLIENT_URL=http://localhost:5173
```

Run

```bash
npm run dev
```

## Features

- Authentication
- Upload CSV / Excel
- Dashboard APIs
- Reports CRUD
- Export PDF / Excel / CSV
- Notifications
- Global Search
- AI Chat
- Forecast

## API Base URL

```
http://localhost:5000/api
```
